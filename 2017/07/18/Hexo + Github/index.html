Hexo + Github 搭建个人博客

发表于 2017-07-18

环境搭建

申请github(这个是必须有的)

创建仓库 仓库名称为 Github昵称.github.io

安装Node.js 英文地址 、中文地址

作用：用来生成静态页面

安装git 官网 ##　　　

作用：把本地的hexo内容提交到github上去.

安装hexo

npm install hexo -g

初始化本地服务

在电脑任意位置创建一个文件夹并打开，然后打开命令行，执行 hexo init, 然后执行 npm install 安装依赖包。

生成静态页面

hexo generate 或 hexo g
hexo server 或 hexo s 启动服务器，可以先看看效果.

配置github

打开文件夹里的配置文件_config.yml，拉到最下面.
修改配置　：后面必须要有空格，否则会出错

deploy:
    　　type: git
    　　repository: https://github.com/x1nGoo/x1nGoo.github.io (这里填写刚创建的仓库地址)
    　　branch: master

执行如下命才能使用git部署(只需要执行一次)　　

npm install hexo-deployer-git --save
执行命令进行部署 hexo deploy 或 hexo d
现在在浏览器地址输入 x1nGoo.github.io 就可以访问到啦

备注

每次部署的步骤 

hexo clean  
hexo generate
hexo deploy

一些常用命令 

hexo new "postName" #新建文章
hexo new page "pageName" #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help  #查看帮助
hexo version  #查看Hexo的版本

文章路径

source/_posts下新建以.md为结尾的文件 

 文章开头格式
---
title: 填写标题
tags: 
- tag1
- tag2
date: 2016/6/06 12:20:25
reward: 是否支持打赏,填写true或者false
keywords: 关键词1 关键词2...
description: 填写描述
---
